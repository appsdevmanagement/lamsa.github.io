Lamsa App – Account & Data Deletion

This page explains how users of the Lamsa app can request deletion of their account and associated data.

How to request account deletion

To request deletion of your Lamsa account and user data, please contact us via WhatsApp using the number below.

  • Send a message requesting Account Deletion
  • Include the email address used to sign in with Google

Data that will be deleted

  • User account information
  • Google account identifier (UID)
  • Email address associated with the account
  • Any app-related user data stored by Lamsa

Data retention

  • No personal user data is retained after account deletion
  • Minimal technical logs may be retained temporarily for security or legal purposes

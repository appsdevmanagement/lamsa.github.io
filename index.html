Account and User Data Deletion

This page explains how users can request deletion of their account and associated data for applications managed by Apps Dev Management.

How to request account deletion

You can request deletion using one of the following methods:

  • Email: apps.dev.management@gmail.com
  • WhatsApp: +96478155143953

Please include the following information in your request:

  • Email address used to sign in (Google account)
  • App name
  • Subject: Account Deletion Request

Data that will be deleted

  • User account identifier (Google UID)
  • Email address associated with the account
  • Any app-related user data stored on our servers

Data retention

  • No personal data is retained after deletion
  • Minimal server logs may be kept for up to 30 days for security and legal purposes

Deletion timeframe

All valid deletion requests are processed within 7 business days.

If you have any questions, contact us using the details above.
